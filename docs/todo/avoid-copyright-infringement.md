# 改版避免侵權原作

## 目標

遊戲玩法與場地佈局變更，避免侵權原作。

---

## 改版對照表

| 項目 | 原版 | 改版 |
|------|------|------|
| **玩家人數** | 3 人 | 1 人 |
| **玩家生命** | 每人 2 HP | 5 HP（血條顯示） |
| **敵人人數** | 初始 3 人，每關 +2 | 不變 |
| **敵人生命** | 每人 3 HP | 不變 |
| **勝利條件** | 擊殺所有敵方 | 命中總次數達到敵方人數 |
| **場地佈局** | 對角線（玩家右下，敵人左上） | 左右對分（敵人左邊，玩家右邊） |
| **玩家移動範圍** | 右下區域 | 右半邊場地 |
| **敵人移動範圍** | 左上區域 | 左半邊場地 |
| **掩體配置** | 雙方各有雪堆 | 玩家側有一個掩體 |
| **敵人 AI** | 隨機移動 + 隨機目標投擲 | 隨機移動 + 集中攻擊唯一玩家（保留投擲誤差） |

---

## 詳細說明

### 1. 玩家設定
- 人數從 3 人改為 1 人
- 生命值：5 次（被命中 5 次才會輸）
- 生命顯示：使用血條
- 移動範圍：右半邊場地自由移動

### 2. 敵人設定
- 人數：初始 3 人，每關 +2（不變）
- 生命值：3 HP（不變）
- 被命中後：僵直狀態（不變）
- 被命中 3 次：死亡（不變）
- 血條：顯示剩餘生命（不變）
- 移動範圍：左半邊場地
- AI 行為：
  - 移動：隨機移動（不變）
  - 投擲目標：集中攻擊唯一玩家（原版是隨機選擇目標）
  - 投擲角度誤差：保留

### 3. 勝利條件
- **原版**：擊殺所有敵方
- **改版**：命中總次數 ≥ 敵方總人數

**範例**：
- 敵方 3 人 → 命中 3 次即過關
- 可能情況 A：命中 A、B、C 各 1 次 = 3 次 = 過關（敵人都沒死）
- 可能情況 B：命中 A 三次 = A 死亡 + 3 次命中 = 過關

### 4. 場地佈局
- **原版**：對角線配置（玩家右下 vs 敵人左上）
- **改版**：左右對分（視覺上分開）
  - 左邊：敵人區域
  - 右邊：玩家區域
  - 雪球可跨越中線攻擊對方

### 5. 掩體配置
- 保留掩體機制
- 玩家側：面對敵人三等分放置

### 6. UI 新增
- 新增「已命中 X / Y 次」顯示過關進度
